◆ targets

bool ecs_metric_desc_t::targets

If id is a (R, *) wildcard and relationship R has the OneOf property, setting this value to true will track individual targets. If the kind is EcsCounterId and the id is a (R, *) wildcard, this value will create a metric per target.
