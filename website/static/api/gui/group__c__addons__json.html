FLECS_API char * ecs_array_to_json ( const ecs_world_t *  world,
ecs_entity_t  type,
const void *  data,
int32_t  count 
)

Serialize array into JSON string. This operation serializes a value of the provided type to a JSON string. The memory pointed to must be large enough to contain a value of the used type.

If count is 0, the function will serialize a single value, not wrapped in array brackets. If count is >= 1, the operation will serialize values to a comma-separated list inside of array brackets.

Parameters
worldThe world.
typeThe type of the value to serialize.
dataThe value to serialize.
countThe number of elements to serialize.
Returns
String with JSON expression, or NULL if failed.
