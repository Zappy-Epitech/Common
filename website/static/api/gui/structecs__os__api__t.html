ecs_os_api_log_t ecs_os_api_t::log_

log callback. The level should be interpreted as: >0: Debug tracing. Only enabled in debug builds. 0: Tracing. Enabled in debug and release builds. -2: Warning. An issue occurred, but the operation was successful. -3: Error. An issue occurred, and the operation was unsuccessful. -4: Fatal. An issue occurred, and the application must quit.
