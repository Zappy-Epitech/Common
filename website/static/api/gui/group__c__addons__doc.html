Detailed Description

Utilities for documenting entities, components, and systems.

Typedef Documentation

◆ EcsDocDescription

typedef struct EcsDocDescription EcsDocDescription

Component that stores description. Used as pair together with the following tags to store entity documentation:

  • EcsName
  • EcsDocBrief
  • EcsDocDetail
  • EcsDocLink
  • EcsDocColor

Get human-readable name from entity. If the entity does not have an explicit human-readable name, this operation will return the entity name.

To test if an entity has a human-readable name, use:
